##介绍
* contentManager是一个我工作早期挣外快为别人开发的网站后台管理程序，可以实现一些管理功能


### 技术选择
* Spring   
* Hibernate   持久层
* Struts2    MVC
* DWR   AJAX框架
* JBPM 工作流
* YUI
* Mysql 数据库

###如何运行
* 拷贝init/content.xml到tomcat的conf目录
* 执行init/back.sql来创建基础数据
* 修改src/resource/config/jdbc.properties来配置数据库信息
* 用eclipse导入工程然后运行tomcat
* 搞定
